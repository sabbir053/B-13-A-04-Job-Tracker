## Answers to Questions ----- >>

### 1. What is the difference between getElementById, getElementsByClassName, and querySelector / querySelectorAll?
	*Ans: getElementById: Return a specific single html element id
		  getElementsByClassName: Returns a collection of all HTML elements that have the specified class name.
		  querySelector: Return just 1st element That match css selector
		  querySelectorAll: returns all elements matching The css selector.

### 2. How do you create and insert a new element into the DOM?
	*Ans: 1st of all catch element to id or classname then create element then use innerHTML and innerText to add content and
			then append to parent in element.

### 3. What is Event Bubbling? And how does it work?
	*Ans: Event Bubbling is when an event (like a click) first on the target element and then bubbles up through its parent 
			elements in the DOM.
			This means that both the child and parent elements can be worked by the same click.

### 4. What is Event Delegation in JavaScript? Why is it useful?
	*Ans: when we attatched a event listener in parent element and then handle it in chield element. Its Properly worrk in bubble
			handling mathod

### 5. What is the difference between preventDefault() and stopPropagation() methods?
	*Ans: preventDefault() stops the default browser action for in a event.

			stopPropagation() stops the event from bubbling up to parent elements.
